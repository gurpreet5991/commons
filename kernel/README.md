## Parent project of Kernel.
This folder has all submodules of kernel of Mosip.

### Sonar Code Analysis 
1. `mvn clean install`

2. `mvn jacoco:report`

3. For local: `mvn sonar:sonar -PLOCAL`

   For dev: `mvn sonar:sonar -PDEV`
